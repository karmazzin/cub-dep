# AGENTS.md

## Правила проекта

- Делай изменения небольшими и строго в рамках запроса.
- Пиши или изменяй код только когда пользователь явно просит об этом.
- Перед изменениями геймплея, UX, генерации мира, блоков, прогрессии или архитектуры читай `SPEC.md`.
- Если запрос на изменение вводит что-то новое относительно `SPEC.md`, обнови `SPEC.md` в той же задаче.
- Держи `SPEC.md` в соответствии с реализованным поведением: нереализованные идеи помечай как запланированные или не реализованные.
- Не переименовывай публичные игровые понятия, id блоков, глобальные объекты или структуру файлов без одновременного обновления кода и `SPEC.md`.

## Стиль кода

- Это статическая браузерная игра с последовательной загрузкой скриптов из `index.html`.
- Глобальное пространство игры: `window.CubDep`.
- Следуй текущему модульному паттерну: IIFE-файлы, которые добавляют API в `Game`.
- Не добавляй зависимости в JavaScript и CSS, если пользователь явно этого не просит.
- Используй ASCII в коде и документации, кроме случаев, где уже нужен русский текст или UI-копирайтинг.

## Проверка

- После изменений кода выполняй релевантную smoke-проверку, например поиск переименованных символов или синтаксически чувствительных ссылок.
- Для изменений UI или геймплея по возможности не проверяй ожидаемое поведение в браузере сам, попроси об этом меня.
- Если для изменения нет автоматического теста, укажи, что было проверено вручную или статическим просмотром.
